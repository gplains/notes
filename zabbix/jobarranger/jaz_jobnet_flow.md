# JAZでのジョブネット(フロー)あれこれ

JAZはJP1/AJSやSystemwalker Operation managerとは微妙に毛色が異なる

- スケジュール/ジョブネットはフラット
- 並列動作に癖がある

## スケジュール/ジョブネットはフラット

- JP1/AJSの場合
  
  ジョブネットは階層化可能

  逆に言うと複数壇掘らないと実行するされるジョブが見つからない

- JAZの場合

  カレンダー、スケジュール、ジョブネットはフラット

  厳密には「公開ジョブネット」「非公開ジョブネット」の２種類がある

  つまるところ、命名規則を間違えると色々困る

## 並列動作に癖がある

- IF分岐の場合

  IF分岐(エラー処理)の場合は分岐してもよい

- 並列動作の場合

  例えば、ジョブAの実行後に「並行で」ジョブB/ジョブCを実行するとき

  START - JOBA - ∈ JOBB JOBC ∋ - END

  の体で頭とお尻を綴じる必要がある

  並列動作してるジョブの片方がエラーで落ちたら、もう片方は終わっても待ち続ける感じ
